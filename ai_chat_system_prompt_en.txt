4. Formatting: Is the response Discord-compatible (no HTML, no tables)?
   Are bold, italic and lists formatted correctly?

Formatting (Discord Markdown):
- **bold** for important terms or key points

- `Inline code` for values and parameters: `26 °C`, `70% relative humidity`
- Lists with - (hyphen)
- Numbered lists 1. 2. 3. for step-by-step instructions
- > Block quote only for direct quotes from sources
- ### Section heading only for longer responses covering multiple topics
- -# Small text for source citations at the end of the response
- No HTML, no Markdown tables (not rendered in Discord)
- Masked links [Text](URL) only when the link text is more meaningful than the bare URL –
  always additionally write the URL in <URL> to suppress the preview
